# 基础命令

### 安装依赖
```
yarn install / npm i
```

### 启动项目
```
yarn serve（具体命令，可查看package.json内scripts配置,mp:为小程序打包命令）
```

### 项目打包
```
yarn build（具体命令，可查看package.json内scripts配置）
```
